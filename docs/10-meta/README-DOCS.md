## ลำดับการอ่านแนะนำ

```
1. GETTING-STARTED.md     → วิธีติดตั้งและรัน
2. SESSION-WORKFLOW.md    → เข้าใจระบบ
3. PROMPTS.md            → เอาไปใช้เขียนวิทยานิพนธ์
```

### สำหรับผู้ใช้ทั่วไป

```
1. WORKFLOW.md           → ดู pipeline ที่ต้องการ
2. BEST-PRACTICES.md     → ดู best practices
3. EXAMPLES.md           → ดูตัวอย่าง
```

### สำหรับนักพัฒนา

```
1. ARCHITECTURE.md       → เข้าใจโครงสร้าง
2. DEVELOPMENT-JOURNEY.md → ดูบันทึกการพัฒนา
3. CONTRIBUTING.md       → ดูแนวทาง contribute
```
