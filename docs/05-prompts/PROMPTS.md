## 1. แม่แบบ Master Prompt

## 2. กฎการเขียน

### 2.1 ภาษาและรูปแบบ
- ใช้ **ภาษาไทยวิชาการ** ตามหลัก ราชบัณฑิตยสถาน
- สรรพนาม: **"ผู้วิจัย"** (ไม่ใช้ "ผม", "ฉัน", "เรา")
- ระดับภาษา: Formal academic register
- เว้นวรรคหลังเครื่องหมายวรรคตอนทุกครั้ง
- ตัวเลข: ใช้ตัวเลขอารบิก ยกเว้นจำนวน 1-10 ที่อยู่ต้นประโยคให้เขียนเป็นคำ

### 2.2 การอ้างอิง
- รูปแบบ: **APA นาม-ปี** (ไม่ใช้ APA ตัวเลข)
- ระบบปี: **พ.ศ.** (พุทธศักราช) เป็นค่าเริ่มต้น
- รูปแบบ in-text citation:
  - แบบประโยค: `ผู้แต่ง (พ.ศ.)` เช่น "สมชาย (2565) กล่าวว่า..."
  - แบบวงเล็บ: `(ผู้แต่ง, พ.ศ.)` เช่น "การวิจัยพบว่า... (สมชาย, 2565)"
- **ห้ามปลอมแปลงชื่อผู้แต่ง ปี หรือแหล่งอ้างอิง** — หากไม่มีข้อมูล ใช้ `[CITATION NEEDED]`

### 2.3 กฎ De-AI
- ค่า CV (Coefficient of Variation) ของความยาวประโยคต้อง >= 30%
- **ห้ามใช้คำและรูปแบบต่อไปนี้:**
  - "นอกจากนี้" ซ้ำเกิน 3 ครั้งต่อบท
  - "ก้าวสำคัญ", "พลิกโฉม", "ล้ำสมัย"
  - ประโยคปิดท้ายว่างเปล่า
- ผสมประโยคสั้น กลาง ยาว อย่างสมดุล

## 3. Prompt สำหรับ De-AI

### 3.1 De-AI Scan (Detect Mode)

```markdown
# ตรวจสอบ AI Writing Patterns

ใช้ skill: thai-de-ai

## ข้อความที่ต้องตรวจสอบ:
{วางข้อความที่ต้องการตรวจ}

## ให้ตรวจสอบ:
1. **Banned Patterns**: คำเชื่อมซ้ำ, ภาษาโฆษณา, ประโยคปิดว่าง
2. **Burstiness (CV)**: ค่า CV ต้อง >= 30
3. **Register**: ระดับภาษา formal/academic
4. **Typography**: เว้นวรรคภาษาไทย

## ภาษา: {thai / english / both}

## รูปแบบผลลัพธ์:
- รายการปัญหาที่พบ
- คะแนน CV
- ข้อเสนอแนะการแก้ไข
```

### 4.1 Thesis Review

```markdown
# ตรวจสอบคุณภาพวิทยานิพนธ์

ใช้ skill: thesis-quality-gate

## มุมมอง: {advisor / committee / iso_auditor}

## เนื้อหาที่ต้องการตรวจสอบ:
{วางเนื้อหาที่ต้องการตรวจสอบ}

## ให้ตรวจสอบ:
1. **โครงสร้าง**: ครบถ้วนตามเกณฑ์
2. **เนื้อหา**: มีความลึกซึ้ง, มีเหตุผล
3. **การอ้างอิง**: ถูกต้องตาม APA
4. **ภาษา**: เป็นไปตามระดับวิชาการ

## รูปแบบผลลัพธ์:
- ข้อคิดเห็น
- ข้อเสนอแนะการแก้ไข
- คะแนน (ถ้ามี)
```

### 4.2 Thesis Score

```markdown

## เนื้อหาที่ต้องการประเมิน:
{วางเนื้อหาที่ต้องการให้คะแนน}

## มิติการให้คะแนน (100 คะแนน):
1. โครงสร้าง (15%): ความครบถ้วนของบท
2. เนื้อหา (25%): ความลึกซึ้งของเนื้อหา
3. ภาษา (20%): ความถูกต้องของภาษา
4. อ้างอิง (20%): ความถูกต้องของอ้างอิง
5. รูปแบบ (15%): ตรง template
6. ความสอดคล้อง (5%): ความสอดคล้องข้ามเอกสาร
```
