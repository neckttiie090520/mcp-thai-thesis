## ทำความรู้จัก AI Writing

### ทำไมต้องลบ?

1. **มหาวิทยาลัยตรวจ** - บางที่ใช้ AI detection
2. **เนื้อหาดูไม่เป็นธรรมชาติ** - ประโยคซ้ำๆ
3. **ภาษาโฆษณา** - ใช้คำเกินจริง

### AI Writing มีลักษณะอะไร?

```
❌ AI Writing:
- ประโยคยาวเหมือนกันหมด
- คำเชื่อมซ้ำ (นอกจากนี้, นอกจากนั้น)
- ภาษาโฆษณา (ก้าวสำคัญ, พลิกโฉม)
- ขาด personal touch

✓ Human Writing:
- ประโยคหลากหลาย (สั้น-กลาง-ยาว)
- คำเชื่อมหลากหลาย
- ภาษาวิชาการเฉยๆ
- มี anecdote/hedge
```

## ทำความรู้จัก CV (Coefficient of Variation)

### สูตร

```
CV = (Standard Deviation / Mean) × 100
```

### เกณฑ์

| CV | ผลลัพธ์ |
|----|----------|
| >= 30% | Human-like ✓ |
| < 30% | Likely AI ✗ |

### ตัวอย่าง

```
ประโยคทั้ง 5 ประโยค:
1. "การปรับแต่งระบบช่วยลดความหน่วงได้"
2. "การปรับแต่งระบบช่วยลดความหน่วงได้อย่างมีประสิทธิภาพ"
3. "การปรับแต่งระบบช่วยลดความหน่วงได้อย่างมีประสิทธิภาพสูงสุด"
4. "การปรับแต่งระบบช่วยลดความหน่วงได้อย่างมีประสิทธิภาพอย่างยิ่ง"
5. "การปรับแต่งระบบช่วยลดความหน่วงได้อย่างมีประสิทธิภาพอย่างยิ่งวิเศษ"

→ CV = 15% (AI-written)
```

```
ต่างกัน:
1. "การปรับแต่งช่วยลด latency"
2. "ผู้วิจัยพบว่า latency ลดลง 30%"
3. "ระบบทำงานเร็วขึ้น"
4. "จากการทดสอบพบว่า..."
5. "วิธีนี้ได้ผลดี"

→ CV = 38% (Human-written)
```

## ทำความรู้จัก Banned Patterns

### หมวด 1: คำเชื่อมซ้ำ

| ห้ามใช้ | แทนที่ด้วย |
|----------|------------|
| นอกจากนี้ (เกิน 3 ครั้ง/บท) | อีกประการหนึ่ง |
| นอกจากนั้น | นอกจากนี้ |
| ยิ่งไปกว่านั้น | ยิ่งไปกว่านี้ |
| ทั้งนี้ | โดยเฉพาะอย่างยิ่ง |

### หมวด 2: ภาษาโฆษณา

| ห้ามใช้ | แทนที่ด้วย |
|----------|------------|
| ก้าวสำคัญ | ปรับปรุง, พัฒนา |
| พลิกโฉม | ปรับเปลี่ยน |
| ล้ำสมัย | ทันสมัย |
| เปลี่ยนแปลงโลก | มีผลกระทบ |
| ยกระดับ | พัฒนา |
| พัฒนาก้าวกระโดด | พัฒนาอย่างมีนัยสำคัญ |
| ปฏิวัติ | เป็นนวัตกรรม |
| นวัตกรรมล้ำ | นวัตกรรมใหม่ |
| เหนือกว่า | ดีกว่า |

### หมวด 3: ประโยคปิดว่าง

| ห้ามใช้ | แทนที่ด้วย |
|----------|------------|
| จะส่งผลดีต่ออนาคต | มีแนวโน้มว่าจะ... |
| อนาคตจะเป็นไปในทิศทางที่ดีขึ้น | ... |
| สิ่งนี้จะเปลี่ยนแปลงทุกอย่าง | ... |

## ใช้ /thesis-deai

### Command

```
/thesis-deai mode={mode} language={language}
```

### Parameters

| Parameter | จำเป็น | ค่า | คำอธิบาย |
|-----------|--------|------|-----------|
| `mode` | ✓ | detect, edit, rewrite | โหมดการทำงาน |
| `language` | - | thai, english, both | ภาษาที่ตรวจ |
| `text` | - | ข้อความ | เนื้อหาที่ต้องการตรวจ |

### 3 โหมด

| Mode | หน้าที่ |
|------|--------|
| `detect` | ตรวจอย่างเดียว ไม่แก้ไข |
| `edit` | แก้ไขตามที่ตรวจพบ |
| `rewrite` | เขียนใหม่ทั้งหมด |

---

## ขั้นตอนการใช้

### Step 1: โหลด Skill

```
/skill thai-de-ai
```

### Step 2: Detect (ตรวจอย่างเดียว)

```
/thesis-deai mode=detect language=thai
```

วางเนื้อหาที่ต้องการตรวจตามด้วย

### Step 3: ดูผลลัพธ์

```
ผลการตรวจ:
- CV Score: 18.5% ← ต่ำ (ควร >= 30%)
- Banned Patterns: 4 รายการ
  1. "นอกจากนี้" พบ 5 ครั้ง
  2. "ก้าวสำคัญ" พบ 2 ครั้ง
  3. "พลิกโฉม" พบ 1 ครั้ง
- Register Violations: 2 รายการ
- Typography: 1 รายการ
```

### Step 4: Edit (แก้ไขตามที่ตรวจพบ)

```
/thesis-deai mode=edit language=thai
```

วางเนื้อหา + ระบุปัญหาที่พบ

### Step 5: Rewrite (เขียนใหม่)

```
/thesis-deai mode=rewrite language=thai
```

วางเนื้อหาต้นฉบับ แล้วระบบจะเขียนใหม่
