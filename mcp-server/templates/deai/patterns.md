# De-AI Patterns / รูปแบบการลบลายนิ้วมือ AI

## Overview / ภาพรวม

เอกสารนี้รวบรวมรูปแบบคำและวลีที่มักพบในข้อความที่สร้างโดย AI พร้อมคำแนะนำในการแก้ไข

---

## Thai Patterns / รูปแบบภาษาไทย

### Tier 1: คำที่พบบ่อยที่สุด

| คำที่พบ | คำทดแทน | หมวดหมู่ |
|--------|---------|----------|
| ในยุคดิจิทัล | ในปัจจุบัน, ทุกวันนี้ | marketing |
| ด้วยการ | โดย, ด้วยวิธี | connector |
| โดยการ | ด้วย, ด้วยวิธี | connector |
| มีความจำเป็น | ต้อง, จำเป็นต้อง | formal |
| มีความสำคัญ | สำคัญ | formal |
| เป็นสิ่งจำเป็น | จำเป็น | formal |
| มีบทบาทสำคัญ | สำคัญต่อ | formal |
| ประสบความสำเร็จ | ได้ผล, สำเร็จ | marketing |
| ผลักดันให้ | ทำให้, กระตุ้นให้ | marketing |
| ยกระดับ | พัฒนา, ปรับปรุง | marketing |

### Tier 2: คำที่พบปานกลาง

| คำที่พบ | คำทดแทน | หมวดหมู่ |
|--------|---------|----------|
| ในปัจจุบันนี้ | ทุกวันนี้, เดี๋ยวนี้ | filler |
| ดังนั้นแล้ว | ดังนั้น, ฉะนั้น | filler |
| อย่างไรก็ตาม | แต่, ทว่า | filler |
| นอกจากนี้ยัง | นอกจากนี้, อีกทั้ง | filler |
| เป็นที่ทราบกันดีว่า | ทราบว่า | filler |
| ไม่ต้องสงสัยเลยว่า | แน่นอนว่า | filler |
| เป็นเรื่องที่น่าสนใจว่า | น่าสนใจว่า | filler |
| มีความเป็นมาอันยาวนาน | มีประวัติยาวนาน | formal |
| ได้รับการยอมรับ | เป็นที่ยอมรับ | formal |
| มีความแตกต่าง | ต่าง, แตกต่าง | formal |

### Tier 3: รูปแบบประโยค

| รูปแบบที่พบ | แนวทางแก้ไข |
|------------|------------|
| "ในบทความนี้ เราจะ..." | เขียนใหม่โดยตรง เช่น "บทความนี้อธิบาย..." |
| "สรุปได้ว่า..." | ใช้ "สรุป..." หรือ "กล่าวโดยสรุป..." |
| "เพื่อให้เข้าใจ..." | เขียนใหม่ เช่น "เพื่อความเข้าใจ..." |
| "จะเห็นได้ว่า..." | ตัดออกหรือเขียนใหม่ |

## Detection Methods / วิธีการตรวจจับ

### 1. Burstiness Analysis / การวิเคราะห์ความหลากหลายของประโยค
- วัดความยาวประโยคที่หลากหลาย
- CV < 20% อาจบ่งชี้ว่าเป็น AI

### 2. Pattern Matching / การจับคู่รูปแบบ
- ค้นหาคำและวลีที่พบบ่อย
- นับจำนวนรูปแบบที่พบ

### 3. Register Analysis / การวิเคราะห์ระดับภาษา
- ตรวจสอบความเป็นทางการ
- หลีกเลี่ยงภาษาที่เป็นทางการเกินไป

### 4. Typography Check / การตรวจสอบการพิมพ์
- ตรวจสอบการใช้เว้นวรรค
- ตรวจสอบเครื่องหมายวรรคตอน

## Rewriting Strategies / กลยุทธ์การเขียนใหม่

### Soul Injection / การใส่ "วิญญาณ"
1. เพิ่มประสบการณ์ส่วนตัว
2. ใช้ตัวอย่างที่เฉพาะเจาะจง
3. แสดงความคิดเห็นที่มีเหตุผล

### Variation / ความหลากหลาย
1. ใช้ความยาวประโยคที่แตกต่างกัน
2. ผสมประโยคง่ายและซับซ้อน
3. ใช้คำเชื่อมที่หลากหลาย

### Human Voice / น้ำเสียงมนุษย์
1. ใช้ภาษาที่เป็นธรรมชาติ
2. หลีกเลี่ยงคำที่ซับซ้อนเกินไป
3. แสดงอารมณ์และความรู้สึกเมื่อเหมาะสม
